Responsive Web Design初窥

一句话总结响应式设计就是：针对任意设备，对网页内容进行完美布局的一种显示机制。

开始响应式设计，首先告别px这个绝对度量单位，改用百分百或em这些相对度量单位。

HTML5和CSS3天生在响应式设计上有优势，原因一言以蔽之就是，它允许我们创建更加简介和快速的代码。
